# Test Plan

## Space Advisor

URL - <https://demo.testim.io/>

GOAL - Build out an end-to-end test suite with typescript

## SCOPE

The purpose of this plan is to describe in detail, the scope of testing and the different activities performed therein for Space Advisor.

**Space Advisor** is an innovative intergalactic space travel and leisure web application, giving families and solo travelers the seamless experience of booking a vacation to the planet of their choosing.

### KNOWN ISSUES

1. Authentication is incomplete.
2. Payment options are not a factor.

## DELIVERABLE(S)

1. Test Plan
2. Test Strategy
3. Risk Analysis
4. Critical Path Definition
5. Test Cases
6. Automation Test Framework in Typescript


### Automation Test Suite for the following critical paths (among other scenarios):

1. The "Happy Path" booking submission
2. Destination Selection
3. Date validation - Departure dates; Date Selection
4. Traveler count logic
5. Confirmation receipt
6. Post Confirmation Navigation
